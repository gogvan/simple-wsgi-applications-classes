Пример wsgi программы на ООП
- получить текст (формат json)
    http://localhost/wsgi/?name_action=get_text&page=1
- получить кодированную картинку, base64 (формат json)
    http://localhost/wsgi/?name_action=get_pic&page=2
- показать все серверные переменные (формат json)
    http://localhost/wsgi/?name_action=get_environ
